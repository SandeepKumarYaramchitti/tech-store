Usually we use the following - 

For Header - <div id= "header">

For Main Content - <div id="main">
For Articles -     <div id="article">
For Section  -     <div id="section">

For side navigation - <div id="aside">
For footer - <div id="footer">

HTML5 Symantics:

<header>
<main>
<article>
<section>
<aside>
<Footer>
